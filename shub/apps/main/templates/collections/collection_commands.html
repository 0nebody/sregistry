Usage

Pull the container to your machine (and optionally name custom, or by hash/commit:

singularity pull shub://{{ collection.repo.owner.login }}/{{ collection.repo.name }}
singularity pull --name customname.img shub://{{ collection.repo.owner.login }}/{{ collection.repo.name }}
singularity pull --commit shub://{{ collection.repo.owner.login }}/{{ collection.repo.name }}
singularity pull --hash shub://{{ collection.repo.owner.login }}/{{ collection.repo.name }}

Shell into the container:

singularity shell shub://{{ collection.repo.owner.login }}/{{ collection.repo.name }}

Run the container:


singularity run shub://{{ collection.repo.owner.login }}/{{ collection.repo.name }}
